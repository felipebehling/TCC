Hardware ESP32 (Guia de Compras)

// 03 · HARDWARE & COMPRAS

Qual ESP32 Comprar?

Evite semanas de debug comprando o chip errado ou economizando onde não deve.

Recomendado

ESP32 DevKitC

Chip WROOM-32 (LX6)

A escolha ideal para 90% dos TCCs. Tem o maior suporte comunitário, suporta TLS pesado sem travar e possui Wi-Fi/BT 4.2 nativos.

ESP32-S3

Chip LX7 com Vetores

Mais poderoso. Excelente se o seu TCC envolver Machine Learning (TinyML) na borda ou necessitar de USB Nativo.

ESP32-C3

Chip RISC-V (Single Core)

Muito barato, substituto do ESP8266. Evite para projetos de segurança, pois o single-core sofre com Handshake MQTTS pesado.

Kit Essencial (Compre Primeiro)

  • 2x ESP32 NodeMCU: Compre dois. Um para enviar dados, outro para interceptar (ou caso frite o primeiro).
  • Protoboard (830 furos) + Jumpers: Kits de cabos M-M e M-F.
  • Sensor BME280: Temperatura/Pressão via I2C. Precisão industrial. Evite o DHT11 (muito impreciso).

O Segredo do Hardware

Analisador Lógico 24MHz 8Ch

Custa ~R$40 no Mercado Livre (Clone Saleae). Ele permite ver visualmente na tela do PC se o dado elétrico está saindo do pino (usando o software grátis PulseView). Vai te salvar semanas de debug cego.

Onde comprar no Brasil com segurança?
FilipeFlop / MakerHero Qualidade garantida e envio rápido. Ótima para sensores oficiais.
RoboCore Curadoria excelente, foco em robótica e IoT de qualidade.
Mercado Livre (Full) Filtre por envios Full. Atenção a placas suspeitas muito baratas.

// 04 · AMBIENTE DE DESENVOLVIMENTO

Abandonando o Amadorismo

Como configurar VS Code, PlatformIO e Git no Linux Manjaro.

Por que o VS Code + PlatformIO?

A IDE do Arduino é excelente para testes de 5 minutos, mas para um projeto de 18 meses com dezenas de bibliotecas e integração de arquivos, o PlatformIO muda as regras do jogo.

  • Gestão via platformio.ini:

    Baixa bibliotecas automaticamente. Sem conflitos de versões de PC para PC.

  • IntelliSense C++ Real:

    Auto-complete poderoso que entende o hardware do ESP32.

  • Controle de Versão:

    O Git funciona dentro da própria janela da IDE perfeitamente.

bash — setup manjaro linux
# 1. Permissões de Serial (Crucial no Linux)
user@manjaro:~$ sudo usermod -a -G uucp $USER
user@manjaro:~$ sudo usermod -a -G dialout $USER
# REINICIE O SISTEMA AGORA

# 2. Instalar VS Code (AUR)
user@manjaro:~$ yay -S visual-studio-code-bin

# 3. No VS Code, instale a extensão PlatformIO IDE.
Git — Obrigatório desde o Dia 1

"Estava funcionando ontem, mudei uma linha e quebrou tudo". Aprenda Git. Crie um repositório privado no GitHub hoje.

git init
git add .
git commit -m "add bme280 i2c"
git push origin main
git checkout -b feature/wifi

// 06 · DOCUMENTAÇÃO & GESTÃO DE PROJETO

Engenharia Exige Escrita

Um projeto brilhante mal documentado resulta em nota medíocre. Blinde seu tempo com processos claros.

Diário (Obsidian) e Kanban

Daqui a 8 meses, você não vai lembrar por que resolveu trocar um pino ou como corrigiu um erro do Wireshark. Use o Obsidian (Markdown offline) para criar sua base de conhecimento ligada.

# Log: 15/10/2024
- Meta: TLS no PubSubClient (Porta 8883)
- Erro: MbedTLS -0x2700 (X509 verify fail)
- Fix: String C++ exige \n no final de cada linha da chave. Link stackoverflow.com/...

Para controle de tempo e tarefas, tenha um Trello/Notion simples: Backlog → Em Progresso → Concluído. Reserve 1 hora no dia 1º de cada mês para organizar o Kanban.

Como sair do bloqueio

  • 01
    Leia o erro inteiro. O compilador de C++ diz a linha exata. Não tenha medo das letras vermelhas.
  • 02
    Isolamento de Variáveis. Crie um arquivo novo limpo. Teste só a função que está falhando, sem o resto do sistema.
  • 03
    A Regra das 2 Horas. Travou por 2h seguidas? Vá dormir ou caminhar. O modo difuso do cérebro resolve o bug no banho.

LaTeX (Overleaf) vs Word

Pule a frustração da formatação ABNT manual.

Citações (BibTeX) Copie o código BibTeX do Google Scholar e o LaTeX constrói sua bibliografia ABNT e ordena tudo automaticamente. Ferramenta útil: Zotero.
Código Limpo Use o pacote listings. O código fonte (C++, Python) sai formatado como na IDE dentro do PDF, sem quebrar layout.
A Estrutura 1. Intro. 2. Ref. Teórico (MQTT, CIA, IEC62443). 3. Metodologia. 4. Dev (O Ataque). 5. Resultados (O TLS e os Gráficos). 6. Conclusão.
